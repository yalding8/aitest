# AI能力测试系统 - 测试操作指南

本文档旨在为测试人员及管理员提供系统的操作指引，涵盖授权码管理、考试流程测试及数据回收。

## 1. 🌐 系统访问

- **考试链接**: `https://test.pylosy.com/`
- **适用设备**: 电脑端（推荐 Chrome/Edge 浏览器）或 手机端。

## 2. 🔑 授权码机制（新）

系统已启用全新的授权码规则，分为**一次性城市码**和**永久测试码**。

### 2.1 永久测试码
*   **代码**: `UHOMES999`
*   **用途**: 供内部测试人员反复验证流程使用。
*   **特性**: 不会被标记为“已使用”，可无限次登录。系统仍会记录每次的测试数据。

### 2.2 正式候选人授权码
格式为 `UHOMES` + `城市简称` + `3位随机数字`。例如：
*   **天津**: `UHOMESTJ123`
*   **北京**: `UHOMESBJ456`
*   **郑州**: `UHOMESZZ789`
*   **成都**: `UHOMESCD012`

> **注意**: 正式码为**一次性使用**。考生提交试卷后，该码立即失效。

## 3. 📝 考试流程测试

1.  **登录**:
    *   输入授权码（如 `UHOMES999`）。
    *   填写个人信息（姓名、邮箱、岗位）。
    *   *注：岗位将影响部分题目的抽取逻辑（咨询/BD/大客户）。*
2.  **答题**:
    *   共 30 道题，限时 60 分钟。
    *   包含单选、多选及判断题。
3.  **提交**:
    *   完成答题后点击提交，或时间结束后自动提交。
    *   提交后系统自动计算及格情况（≥70分及格）并不予显示具体分数。

## 4. 📊 数据查询与导出

测试人员或管理员可通过服务器上的脚本导出完整的使用记录。

### 4.1 使用一键导出工具

在项目根目录下运行以下脚本（需具备服务器 SSH 权限）：

```bash
./manager-export.sh
```

### 4.2 查收报表

执行成功后，本地文件夹将生成两份 Excel 可读的 CSV 文件：

1.  **授权码_已使用_YYYYMMDD.csv**:
    *   包含：授权码、城市、使用时间、**考生姓名、邮箱、岗位、分数**。
2.  **授权码_未使用_YYYYMMDD.csv**:
    *   包含：剩余可用的授权码清单。

---

### 💡 常见问题 (FAQ)

*   **Q: 考生反馈“授权码无效”？**
    *   A: 请确认授权码是否已被使用，或是否有输入错误（不区分大小写）。
*   **Q: 如何新增更多授权码？**
    *   A: 需在服务器端生成新的 JSON 数据并覆盖 `授权码.json`。
