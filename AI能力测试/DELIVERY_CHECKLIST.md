### 🔧 后端层（第三阶段）

- [x] **Express 服务器** (`server.js`) - 完整实现
  - CORS 跨域支持
  - JSON 请求体解析

- [x] **三个核心 API**

  1. **POST /api/verify**
     - 验证授权码有效性
     - 检查授权码未被使用
     - 返回验证结果

  2. **POST /api/start**
     - 生成唯一的 examId
     - **随机抽取30道题**
       - 按维度分层（AI认知8 + 场景10 + 思维7 + 实战5）
       - 再次随机打乱顺序
     - 保存考试数据（内存存储）
     - 返回题目列表（不包含答案）

  3. **POST /api/submit**
     - 接收用户答案
     - **自动计分**
       - 对比标准答案
       - 支持多选题判断
       - 分数标准化到100分
     - **标记授权码为已使用**
     - **Webhook 推送结果**
     - 清理内存

- [x] **核心算法**

  - **随机出题算法**：按维度分层随机
    ```
    分层抽取：AI工具认知 8道 + 岗位应用 10道 + 思维转变 7道 + 实战判断 5道
    再次打乱：sort(() => Math.random() - 0.5)
    ```

  - **计分算法**：标准化评分
    ```
    逐题判断 -> 累积得分 -> 标准化到100分 -> 与80分比较
    ```

  - **授权码管理**：一次性使用
    ```
    验证时检查 status !== 'used'
    提交时自动标记为 'used' + 时间戳
    ```

- [x] **Webhook 推送**
  - 飞书 Webhook 格式
  - 推送内容：姓名、邮箱、岗位、分数、状态、用时
  - 错误处理和日志记录
