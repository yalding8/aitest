# 岗位差异化题目设计方案

## 一、岗位能力模型

### 咨询顾问
**核心能力**：
- 客户沟通与需求理解
- 问题快速响应
- 租房流程指导
- 投诉处理
- 多语言服务

**题目侧重**：
- 客户服务场景（高）
- 多语言沟通（高）
- 实战判断（中）
- 流程优化（中）

### 渠道BD
**核心能力**：
- 商务拓展与谈判
- 房东/合作伙伴开发
- 数据分析与市场洞察
- 资源整合
- 创新合作模式

**题目侧重**：
- 岗位场景应用（高）
- 数据分析（高）
- 流程优化（中）
- 创新应用（中）

### 大客户经理(KA)
**核心能力**：
- 大客户关系维护
- 方案设计与呈现
- 批量订单管理
- 风险控制
- 数据分析与报告

**题目侧重**：
- 岗位场景应用（高）
- 数据分析（高）
- 风险控制（高）
- 客户服务（中）

## 二、题目分配策略

### 通用题目（60题）
所有岗位都会抽取的基础题目：
- AI工具认知：8题（全部通用）
- 思维转变识别：8题（全部通用）
- 实战判断：6题（全部通用）
- 多语言沟通：8题（全部通用）
- 客户服务：10题（通用）
- 数据分析：6题（通用）
- 流程优化：6题（通用）
- 风险控制：5题（通用）
- 创新应用：3题（通用）

### 岗位专属题目（40题）
每个岗位10-15题专属：

**咨询顾问专属（15题）**：
- 客户服务：10题（紧急情况、投诉处理、个性化服务）
- 多语言沟通：5题（复杂翻译场景）

**渠道BD专属（15题）**：
- 岗位场景应用：10题（房东开发、商务谈判）
- 数据分析：3题（市场分析、ROI计算）
- 创新应用：2题（新渠道探索）

**大客户经理专属（15题）**：
- 岗位场景应用：10题（大客户方案、批量管理）
- 数据分析：3题（客户分析、预测）
- 风险控制：2题（合同审核、合规）

## 三、抽题规则调整

### 咨询顾问（30题）
- AI工具认知：3题（通用）
- 思维转变识别：2题（通用）
- 实战判断：2题（通用）
- 多语言沟通：4题（3通用+1专属）
- 客户服务：10题（5通用+5专属）
- 数据分析：2题（通用）
- 流程优化：3题（通用）
- 风险控制：2题（通用）
- 岗位场景应用：2题（通用）

### 渠道BD（30题）
- AI工具认知：3题（通用）
- 思维转变识别：2题（通用）
- 实战判断：2题（通用）
- 多语言沟通：2题（通用）
- 客户服务：4题（通用）
- 数据分析：5题（3通用+2专属）
- 流程优化：3题（通用）
- 风险控制：2题（通用）
- 岗位场景应用：5题（2通用+3专属）
- 创新应用：2题（1通用+1专属）

### 大客户经理（30题）
- AI工具认知：3题（通用）
- 思维转变识别：2题（通用）
- 实战判断：3题（通用）
- 多语言沟通：2题（通用）
- 客户服务：5题（通用）
- 数据分析：5题（3通用+2专属）
- 流程优化：3题（通用）
- 风险控制：3题（2通用+1专属）
- 岗位场景应用：4题（1通用+3专属）

## 四、实施步骤

4. 测试验证
